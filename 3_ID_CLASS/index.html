ID e CLASS

ID

O ID funciona com um CPF. Assim como so posso ter um numero de CPF, cada elemento so pode ter um valor de ID. O CPF é único, ou seja, ninguem pode ter um CPF igual ao meu. Isso significa que o ID tambem é única, entã nao e possivel repertir o mesmo valor duas vezes.

CLASS

O CLASS funciona de maneira oposta ao ID. Diferente do ID em que voce so pode ter um valor, no CLASS voce pode ter varias valores. Tambem diferente do ID, em que nao se pode repertir um valor, no CLASS voce pode ultilizar o mesmo valor para varios elementos
